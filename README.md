我本停驻丹蔷花海恣意轮舞尖刺间放旷

又有绝美幻象邀我共享这鸩酿

用曾时构建过简易乌托邦 换刻庭暖花香

便算风中耳语也渐变温热悠扬

当极夜降至温宁这刹风光 或戏文草收场

谁难得凭真实感想领意念疏狂

我仍可笑容天真不动声响 舐温血以弥伤

凛冽凌乱尽头或有休憩眠月港
